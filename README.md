# AgroCineTickets

## Description

Microservices-based backend application to manage a ticket sale.

## How-To

### Requirements

- [Docker Compose][docker-compose]

### Running

```shell
    $ git clone https://github.com/Iuri-Almeida/AgroCineTickets
    
    $ cd AgroCineTickets
    
    $ docker compose up
```
